# enc

**enc** is a lightweight Python-based file packer/unpacker utility. It bundles multiple files into a single `.enc` archive (a renamed `.zip`) and can decode these archives back to original files easily.

## Features

- Simple command-line encode/decode  
- Uses standard compression for fast and reliable packing  
- Portable `.enc` archive format  

## Usage

# Encode files into archive
enc -encode file1.txt file2.png --name=myarchive.enc

# Decode archive back into files
enc -decode myarchive.enc
